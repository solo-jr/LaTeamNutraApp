ROOTTSMG BACKEND THEME

ROOTTSMG BACKEND THEME is a backend theme module for your odoo community

If you install ROOTTSMG CUSTOM THEME COLOR, this module can customize the background image of the apps menu and these colors

ROOTTSMG BACKEND THEME+ROOTTSMG CUSTOM THEME COLOR

rtmg_custom_theme_color

This module allows you to customize the colors and opacity of the odoo theme, whether it is community or enterprise

rtmg_web_patch

With this module you can personalize the signature of odoo in the website and the title of the back office
